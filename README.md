# Anime_recommender
Machine learning project that recommends similar anime using K-Nearest Neighbors (KNN) and cosine similarity, with an interactive Streamlit frontend.
## Features
- Recommend similar anime based on genres, ratings, and episode count
- Interactive Streamlit web interface
- Data preprocessing with pandas
- Genre one-hot encoding
- Feature normalization using MinMaxScaler

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
